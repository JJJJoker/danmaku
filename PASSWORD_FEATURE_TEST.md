## 🔍 调试技巧

### 查看客户端日志
在客户端控制台中查看:
```javascript
[ServerConnection] Joined room successfully, isHost: true
[ServerConnection] Set password for room: 加密房间
```

### 查看服务器日志
```bash
pm2 logs danmaku-server --lines 50
```
应该看到:
```
[Server] Room created: 加密房间 (1/100)
[Server] Wrong password for room: 加密房间, user: abc123
[Server] Password changed for room: 加密房间, hasPassword: true
```

### 检查网络请求
打开浏览器开发者工具 -> Network -> WS标签
查看WebSocket消息:
```json
// 客户端发送
{"type":"join","payload":{"roomId":"加密房间","userId":"abc123","username":"张三","password":"123456"}}

// 服务器响应
{"type":"joinSuccess","payload":{"roomId":"加密房间","userId":"abc123","isHost":true}}
```

## ⚠️ 已知限制

1. **房主身份不持久化**: 当房主离开房间后重新加入,将失去房主身份
2. **明文传输**: 密码以明文形式在WebSocket中传输(未来可使用WSS加密)
3. **无密码强度检查**: 允许任意长度的密码(建议至少6位)
4. **无速率限制**: 可以频繁修改密码(未来可添加限流)
